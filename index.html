Top 500 songs since 2009

Using Kimono, I gathered songs that made it to the top 100 annual chart since 2009. I extracted additional information – such as gender of the artist(s) and song lyrics – using multiple APIs, including last.fm and lyricsnmusic. I filtered and cleaned the data using Python. The data is now located on Google Spreadsheets and is extracted thanks to tabletop.js and sheetsee.js. It is visualized using D3.js. Some visualizations were easy to implement, while others more challenging. This assignment was exciting, informative and an absolute delight.
